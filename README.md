## Getting Started
Choose one of these two options:
- Run with dotnet CLI.
- Run with docker compose

### Dotnet CLI
```zsh
dotnet restore
dotnet build
dotnet run
```

### Run with Docker compose
```zsh
 docker-compose build
 docker-compose up
```
